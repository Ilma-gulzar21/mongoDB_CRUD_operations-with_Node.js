  <h1>MongoDB CRUD Operations with Node.js</h1>

  <h2>Description</h2>
  <p>
    This project is created to practice MongoDB with Node.js.
    In this project, CRUD operations (Create, Read, Update, Delete)
    are performed using Node.js to understand database connectivity
    and data handling.
  </p>

  <h2>Features</h2>
  <ul>
    <li>MongoDB connection with Node.js</li>
    <li>Create data in MongoDB</li>
    <li>Read data from MongoDB</li>
    <li>Update existing data</li>
    <li>Delete data from MongoDB</li>
    <li>Backend practice project</li>
  </ul>

  <h2> Technologies Used</h2>
  <ul>
    <li>Node.js</li>
    <li>MongoDB</li>
    <li>Mongoose</li>
  </ul>

  <h2>CRUD Operations</h2>
  <ul>
    <li><strong>Create:</strong> Add new data to the database</li>
    <li><strong>Read:</strong> Fetch data from the database</li>
    <li><strong>Update:</strong> Update existing records</li>
    <li><strong>Delete:</strong> Remove records from the database</li>
  </ul>

  <h2> How to Run the Project</h2>
  <ol>
    <li>Clone the repository to your local machine:</li>
    <pre>git clone &lt;repository-url&gt;</pre>

    <li>Navigate to the project folder:</li>
    <pre>cd project-folder</pre>

    <li>Install the required dependencies:</li>
    <pre>npm install</pre>

    <li>Start MongoDB server:</li>
    <p>
      Make sure MongoDB is installed and added to your system path.  
      Open a new terminal and run:
    </p>
    <pre>mongod</pre>

    <li>Run the Node.js project:</li>
    <pre>node app.js</pre>
  </ol>

  <h2> Learning Purpose</h2>
  <p>
    This project is made for learning and practice purposes only.
    It helps to understand how MongoDB works with Node.js and how
    CRUD operations are implemented.
  </p>
